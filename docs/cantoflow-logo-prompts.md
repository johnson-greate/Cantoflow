No microphone, no speech bubble, no headphones, no robot, no brain, no flags, no Chinese character, no purple-blue gradient, no neon glow, no generic AI sparkle, no mockup device, no photographic texture, no tiny decorative details. Clean flat vector artwork, crisp edges, suitable for export as SVG and transparent PNG, 1024×1024.

1. Generate four variations of Direction A.
2. Select the strongest silhouette and regenerate it as a black/white single-colour mark to test 16 px readability.
3. Use the selected mark as a reference image for final app-icon, menu-bar monochrome, and transparent-background exports.
