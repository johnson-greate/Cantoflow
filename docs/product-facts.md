# CantoFlow Product Facts

Verified: 2026-06-20

## GPT Image 2

- Official API model ID: `gpt-image-2`.
- It is available through the OpenAI Image API for image generation and editing.
- Official guide: https://developers.openai.com/api/docs/guides/image-generation
- CantoFlow currently has no committed logo or app-icon asset; the logo work in this iteration is prompt exploration, not a silent replacement of the production icon.

## DeepSeek V4 Flash

- Official API model ID: `deepseek-v4-flash`.
- OpenAI-compatible base URL: `https://api.deepseek.com`.
- Chat endpoint: `https://api.deepseek.com/chat/completions`.
- Thinking mode is enabled by default. Low-latency text polishing should explicitly send `"thinking": {"type": "disabled"}`.
- Official quick start: https://api-docs.deepseek.com/
- Official model/pricing page: https://api-docs.deepseek.com/quick_start/pricing
- Official chat completion reference: https://api-docs.deepseek.com/api/create-chat-completion

## Capsule Reference

- The supplied Spokenly screenshot is used only as a product-behaviour reference: compact floating feedback, clear status hierarchy, and an audio-reactive waveform.
- CantoFlow should retain its own visual identity rather than reproducing Spokenly's exact shape, colours, or layout.
